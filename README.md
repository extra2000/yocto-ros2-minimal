A minimal ROS2 with Yocto/PetaLinux workspace for learning purpose.


## Getting Started

Clone this repository:
```
git clone --recursive https://github.com/extra2000/yocto-ros2-minimal.git
```

* Xilinx:
    * [Arty Z7-20](docs/petalinux/arty-z7-20.md)
